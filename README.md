## Course Description

Spatial analysis has traditionally required a steep learning curve — tools like QGIS and ArcGIS demand dedicated training and resist adaptation to new workflows. Generative AI and agent-based systems break down this barrier: they dramatically lower the cost of learning geospatial tools and accelerate the construction of analytical pipelines.

This course develops practical applications of AI for spatial data by exploiting the latest resources available through APIs, open-source packages, and freely available models. The goal is to expand access to geospatial tools through generative AI, equipping participants to build real-world solutions for their own projects.

The course is divided into two parts. **Part 1** focuses on geospatial analysis with Python — the most widely used libraries, open data APIs, and data sources from NASA, Google Earth, and ESA. **Part 2** focuses on building functional digital applications: generative AI agents, prompt engineering, and deploying interactive GeoAgent apps in participants' own work environments.

---

## Learning Outcomes

By the end of this course, participants will be able to:

1. **Apply Python** for geospatial analysis — working with vector data (points, lines, polygons, shapefiles) and raster data (GeoTIFF, NetCDF), performing choropleth mapping, heatmaps, spatial joins, and zonal statistics.
2. **Use generative AI** to accelerate spatial analysis workflows through prompt engineering and tool-calling agents.
3. **Build visual tools** with geospatial data enhanced by generative AI to communicate results effectively to non-technical audiences.
4. **Develop and deploy GeoAgent applications** — interactive, AI-powered geospatial tools deployable in real work environments.

Professionals, graduate students, public servants, social researchers, and decision-makers interested in applying large language models to social data analysis and public-sector solutions. Basic programming knowledge and foundational data analysis skills are recommended.

## Course Content

### Part 1 — Python for Spatial Analysis (12 hours · 4 sessions)

**Session 1 · Fundamentals of Python for Spatial Analysis**
- Points, lines, polygons, shapefiles, rasters, and NetCDF files
- Generating choropleth maps and heatmaps
- Key operations: intersection, zonal statistics, spatial joins

**Session 2 · Advanced Spatial Analysis with Python**
- Generating dynamic and interactive maps
- Modern libraries: geowombat, geoapache
- Large spatial databases: geoapache, spatial SQL

**Session 3 · Generating Spatial Data & Climate Models**
- OpenStreetMap — the open georeferenced data library
- Geocoding: Google Maps, Bing Maps
- Data sources: NASA, Google Earth, ESA, Climate Change datasets
- Particle movement modeling: HYSPLIT

**Session 4 · Foundational Spatial Models**
- Transformers for spatial data
- Embeddings for classification
- Land cover classification

---

### Part 2 — Generative AI for Spatial Analysis (12 hours · 4 sessions)

**Session 5 · Fundamentals of AI Agent Development**
- Setup and access to LLM APIs
- Introduction to the DeepAgents LangChain framework
- Simple tool calling

**Session 6 · Introduction to GeoAgents**
- Open source & local AI models
- Multiple tool calling
- Skills

**Session 7 · GeoAgents App Development**
- Memory
- Structured output
- Streamlit

**Session 8 · GeoAgents App Deployment**
- Troubleshooting
- Model Context Protocol (MCP)
- Final project presentations
