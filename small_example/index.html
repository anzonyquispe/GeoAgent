Peru Environmental Analysis

Fire Detection and Deforestation at Municipality Level

Overview

This analysis examines fire activity and forest change across Peru's ~1,874 municipalities (GADM Level-3 districts) using satellite data from NASA and the University of Maryland.

Fire Data

NASA VIIRS SNPP

Active fire detections at 375m resolution from the VIIRS sensor on the Suomi NPP satellite.

Forest Data

Global Forest Watch

Hansen et al. tree cover change data at 30m resolution, covering 2000-2024.

Analysis Components

Fire Analysis

Municipality-level aggregation of fire detections including:

  • Fire count by district
  • Fire Radiative Power (FRP)
  • Temporal patterns (monthly/yearly)
  • Regional hotspots

Forest Analysis

Deforestation metrics by municipality including:

  • Forest cover 2000 (baseline)
  • Annual forest loss 2001-2024
  • Forest gain 2000-2012
  • Net change trends

Methodology

Analysis performed using OpenSourceGeoAgent, a Python class that enables geospatial analysis with open source LLMs (via Ollama). The workflow includes:

  1. Download administrative boundaries from GADM (Level-3: ~1,874 districts)
  2. Fetch fire data from NASA FIRMS API
  3. Download Hansen GFC tiles for forest analysis
  4. Spatial aggregation using GeoPandas
  5. Visualization with Folium and Matplotlib
